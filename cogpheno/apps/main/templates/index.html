CogatPheno is a database of cognitive and behavioral assessments with questions that have been mapped to concepts and contrasts defined in the Cognitive Atlas.
CogatPheno will (eventually) allow for a data-driven derivation of an individual's cognitive phenotype using one or more assessments. Given that the Cognitive Atlas is an ontology, this phenotype can be expressed at multiple levels of detail. We hope to be able to describe currently categorically defined mental illness using these dimensional concepts.
This work is being done by the Poldracklab at Stanford University.
